# CS 374: Database Systems — Assignments

Coursework from CS 374 (Database Systems) at JMU, covering SQL from beginner to advanced, CRUD operations in Python, EER modeling, relational mapping, and a semester-long hotel database project (design, build, and query).

## Contents

| Folder | Assignment | Topics |
|---|---|---|
| `assignment-1-beginning-sql` | HW1: Beginning SQL | SQLite, `JOIN`, `GROUP BY`, `LIMIT`, expressions |
| `assignment-2-crud-operations` | HW2: CRUD Operations | Python DB-API, `CREATE`/`INSERT`/`UPDATE`/`DELETE` |
| `assignment-3-hotel-eer-model` | HW3: Hotel EER Model | Conceptual database design, EER diagramming |
| `assignment-4-eer-to-relational-mapping` | HW4: EER to Relational Mapping | Converting EER model to relational schema |
| `assignment-5-intermediate-sql` | HW5: Intermediate SQL | PostgreSQL, `GROUP BY`/`HAVING`/`LIKE`, outer joins, functions |
| `assignment-6-advanced-sql` | HW6: Advanced SQL | Set operations, window functions |
| `assignment-7-create-populate-hotel-db` | HW7: Create and Populate the Hotel Database | Schema creation, data loading (SQL/Python/Faker) |
| `assignment-8-query-hotel-database` | HW8: Query the Hotel Database | Full CRUD queries, transactions, indexing, views |

## About the Hotel Database Project (HW3, 4, 7, 8)
Assignments 3, 4, 7, and 8 form a connected semester-long project: designing a database for a hotel chain from scratch (EER model), mapping it to a relational schema, building and populating it, and finally writing real-world queries (reservations, check-in/check-out, billing, occupancy history) against it.

## Tools Used
SQLite, PostgreSQL, pgAdmin, DB Browser for SQLite, MySQL Workbench / dbdiagram.io, Python (`sqlite3`, `psycopg`, Faker)
